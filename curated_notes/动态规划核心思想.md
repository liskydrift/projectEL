# 动态规划核心思想

## 三要素
1. **最优子结构**：原问题的最优解包含子问题的最优解
2. **重叠子问题**：子问题被重复计算，可通过记忆化避免
3. **状态转移方程**：描述子问题间的关系

## 思考步骤
1. 定义状态（dp[i] 表示什么）
2. 找到转移方程（dp[i] = ?）
3. 确定初始条件（base case）
4. 确定遍历顺序

## 经典例题
- 斐波那契数列 → 一维 DP
- 0-1 背包问题 → 二维 DP
- 最长公共子序列 → 二维 DP
- 编辑距离 → 二维 DP
